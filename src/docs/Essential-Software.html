1.2.4 Essential Software

Here is a list of software that you might want to have on your system.

BOOTLOADERS

  • efibootmgr
  • grub

FILESYSTEMS

  • e2fsprogs
  • dosfstools
  • ntfs-3g

NETWORKING

  • dhcpcd
  • wpa_supplicant

TEXT EDITORS

  • nano
  • vim
  • neatvi
  • emacs
  • emacs-nox (terminal-only version of emacs)

USER SHELLS

  • bash
  • zsh
  • oksh
  • rc

POSIX BASE UTILITIES

  • busybox
  • sbase
  • coreutils

DOCUMENTATION

  • carbs-docs
  • man-pages
  • man-pages-posix
